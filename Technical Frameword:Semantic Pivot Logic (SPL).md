2. The Logographic Pivot (The Core Innovation)

The SPL architecture rejects probabilistic guessing in favor of Deterministic Mapping. The process involves three distinct phases:

    Phase I: Semantic Deconstruction: The user’s natural language query is stripped of its syntactic ambiguity.

    Phase III: Recursive Reconstruction: The browser DOM is rebuilt using only verified entity anchors. This forces the browser to ignore the AI Overview box and focus on high-fidelity metadata (e.g., the "Starmer" or "Trump" entity windows).

To ensure the SPL remains autonomous, we utilize a State-Partitioning Protocol. This forces the browser's internal ML engine (browser.ml.enable=false) into a quiescent state, preventing it from "hallucinating" over the user's architectural view.

By targeting the temporal and entity-based pivots of a search result, the logic ensures that the user is presented with a clean, high-contrast interface—optimized for human intelligence rather than algorithmic manipulation.
